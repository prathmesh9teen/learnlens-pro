# 🎓 LearnLens Pro  

## 🚀 Overview  
AI-powered dashboard for clustering student behaviour from LMS logs.

## ✨ Features  
- KMeans, Hierarchical, DBSCAN  
- PCA visualization  
- Real-time metrics  
- Interactive UI  

## 🛠️ Tech Stack  
Python, Scikit-learn, Pandas, NumPy, JavaScript, Chart.js  

## ▶️ Run  
Open index.html in browser
