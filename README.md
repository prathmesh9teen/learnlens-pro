## 🧠 Overview  

LearnLens Pro is an **interactive machine learning dashboard** that analyzes LMS interaction data and clusters students into meaningful learning personas using **unsupervised learning algorithms**.

💡 It helps educators:
- Identify student behavior patterns  
- Detect at-risk learners  
- Improve learning outcomes  

---

## ✨ Key Features  

🔹 Real-time clustering simulation  
🔹 Interactive dashboard with animations  
🔹 Multi-algorithm comparison (KMeans, DBSCAN, Hierarchical)  
🔹 PCA-based 2D visualization  
🔹 Live metrics (Silhouette Score, Davies-Bouldin Index)  
🔹 Automatic persona generation
